### 🚀 Featured Project — MonSolde

A bilingual (French/English) budgeting PWA with an AI assistant named **Sol**, built iPhone-first with offline support and home-screen installability.

- 💬 **Sol** — an AI assistant powered by the Claude API, built right into the app
- 📱 iPhone-first PWA: Dynamic Island safe areas, offline-first caching, installable like a native app
- 🇫🇷🇬🇧 Fully bilingual UI from the ground up
- ⚡ Built with React + Vite, backed by Supabase, deployed on Vercel

> Budgeting apps are boring. I wanted to build one I'd actually want to open every day.
